Výhody Open Props

  • modularita - vezmu jen to, co potřebuji
  • pro jakýkoliv framework (css, js)
  • light / dark theme out of the box
  • 0 specificita - pomocí selektoru :where()
  • performance - pokud se vyhneme nějakému hromadnému selectoru typu *, i vyšší počet custom properties by neměl mít na performance vliv

Inspirace pro vlastní design tokens

  • jde to postavit na CSS custom properties
  • geniální systém škál --red-{0-9}
  • neustálý boj s pojmenováním škál
    • z-index: underground / ground / overlay / sticky / menu / modal (originální názvy)
    • font-weight: light / regular / medium / bold / black (ustálené názvy)
    • font-size: xs / sm / base / lg / xl / 2xl / 3xl (Tailwind)
