sneek peek

grid-template-columns ↗

``` .grid-container { display: grid; grid-gap: 15px; /* grid-column-gap a grid-row-gap */ grid-template-columns: auto auto auto; } ```

Libovolné jednotky

``` .grid-container { display: grid; grid-gap: 15px; /* grid-column-gap a grid-row-gap */ grid-template-columns: 150px 70% auto; } ```

novinka: fr units

``` .grid-container { display: grid; grid-gap: 15px; /* grid-column-gap a grid-row-gap */ grid-template-columns: 1fr 3fr 1fr; } ```

funkce repeat() ↗

``` .grid-container { display: grid; grid-gap: 15px; /* grid-column-gap a grid-row-gap */ grid-template-columns: repeat(12, 1fr); } ```

funkce minmax() ↗

``` .grid-container { display: grid; grid-gap: 15px; /* grid-column-gap a grid-row-gap */ grid-template-columns: minmax(100px, 350px) 1fr 1fr; } ```

kombinace funkcí + auto-fill

``` .grid-container { display: grid; grid-gap: 15px; /* grid-column-gap a grid-row-gap */ grid-template-columns: repeat(auto-fill, minmax(150px,1fr)); } ```

grid-template-rows ↗

``` .grid-container { display: grid; grid-gap: 15px; /* grid-column-gap a grid-row-gap */ grid-template-columns: repeat(3, 1fr); grid-template-rows: repeat(3, 100px); } ```

grid-columns-end ↗ / grid-row-end

``` .grid-container { display: grid; grid-gap: 15px; grid-template-columns: repeat(auto-fill, minmax(150px,1fr)); } .span-2-column { grid-column-end: span 2; } .span-2-all { grid-column-end: span 2; grid-row-end: span 2; } ```

order

``` .grid-container { display: grid; grid-gap: 15px; grid-template-columns: repeat(auto-fill, minmax(150px,1fr)); } .span-2-column { grid-column-end: span 2; order: 1; } .span-2-all { grid-column-end: span 2; grid-row-end: span 2; } ```

Co neumí nikdo jiný

grid-template-areas

``` .grid-container { display: grid; grid-gap: 15px; grid-template-areas: "a b b g" "c c d g" "e f d g" } ```
``` .grid-item--a { grid-area: a; } .grid-item--b { grid-area: b; } .grid-item--c { grid-area: c; } .grid-item--d { grid-area: d; } .grid-item--e { grid-area: e; } .grid-item--f { grid-area: f; } .grid-item--g { grid-area: g; } ```
