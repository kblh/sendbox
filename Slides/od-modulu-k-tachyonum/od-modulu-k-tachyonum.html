Problémy utilitárních tookitů

  • neexistuje standard v názvech tříd
    (v-top, v-a:t, align-top)
  • snaží se nahradit modulární metodiky
    (SMACSS, OOCSS, BEM)
  • využijete max 10 % - 20 % tříd

Zkušenosti s Micro Tachyons

  • už jsem ho několikrát použil (Logio)
  • má ještě znaky frameworku (proměnné)
  • nemá standardizované názvy tříd
  • není standard - problematické rozšiřování

EMMET

EMMET

Emmet (formerly Zen Coding) is a set of plug-ins for text editors that allow for high-speed coding and editing in HTML, XML, XSL, and other structured code formats via content assist.

  • umí ho nativně kdejaký editor (Aptana/Eclipse, Atom, VS Code, Notepad++)
  • používá krátké a poměrně srozumitelné zkratky - ideální pro názvy našich subatomárních tříd (dn, posa, mb1r)
  • díky tomu lze knihovnu jednoduše rozšiřovat při zachování konvence pojmenovávání
