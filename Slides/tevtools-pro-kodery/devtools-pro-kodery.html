Chrome DevTools for coders

Keyboard Shortcuts

  • ⌘ + opt + C (Mac) or ctrl + shift + C (Windows, Linux, Chrome OS).
  • ⌘ + opt + J: console
  • ⌘ + shift + D: toggle the dock state between the last two recently used states
  • ⌘ + shift + P: show panel

Workspaces

  • ⌘ + click: go to exact line of code
  • Edit Files With Workspaces

Global search

  • ⌘ + opt + F

Console Utilities

  • $0, $1, $2 ... $4: historical reference to the last five inspected DOM elements
  • $(selector): alias for the document.querySelector() function
  • $$(selector): alias for the document.querySelectorAll() function
  • $_: returns the value of the most recently evaluated expression
  • Console Utilities API Reference

Console Live Expressions

  • Examples: window.pageYOffset, document.location, document.cookie

  • Example: document.designMode = "on"
  • ⌘ + Enter
  • Panel: !
