Méně známé, ale užitečné CSS vlastnosti

Petr Koláček

zoom

``` p.small { zoom: 75%; } p.normal { zoom: normal; } p.big { zoom: 2.5; } ```

background-blend-mode

Example, Doc

``` .card { background-image: linear-gradient(-134deg, #FFEA9D 0%, #FF9458 100%), url(https://unsplash.it/g/300/300?random); background-size: cover; background-position: center; } .-color { background-blend-mode: color; } ```

box-decoration-break

Example

``` .text { background-image: linear-gradient(-144deg, #6AEEC6 0%, #698D28 100%); } .-clone { box-decoration-break: clone; } ```

column-count

Example

``` .columns { column-count: 3; column-gap: 25px; column-rule: 1px solid #454545; } ```

counter

Example

``` body { counter-reset: section; } h3::before { counter-increment: section; content: "Section " counter(section) ": "; } ```

currentColor

Example

``` button { color: red; border: 1px solid currentColor; } button svg { fill: currentColor; }```

table-layout

Example

``` .table--fixed { table-layout: fixed; } ```

max(), min(), clamp()

Example

``` .container { min-width: 300px; width: 50%; max-width: 700px; width: clamp(300px, 50%, 700px); } ```
